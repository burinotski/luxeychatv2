# LuxePrints PH Web Interface

A simple web interface for LuxePrints PH featuring:
- Dark theme
- Three-panel layout
- Chat interface
- Responsive design

## Setup
1. Clone the repository
2. Open index.html in a web browser

## Structure
- `index.html` - Main HTML file
- `css/dark-theme.css` - Dark theme styling
